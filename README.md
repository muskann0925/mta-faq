# MarTechAdda FAQ

A modern, responsive FAQ page built for MarTechAdda using React, TypeScript, and Tailwind CSS. It provides a clean user experience for browsing, searching, and managing frequently asked questions.

## Features
- Responsive Navbar & Footer
- Search FAQs instantly
- Category-based FAQ organization
- Smooth Accordion UI
- Contact Support Modal
- Book a Call Modal
- Mobile-friendly design
- Modern gradient-based UI

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Vite

## Getting Started

Clone the repository:

```bash
git clone https://github.com/muskann0925/mta-faq.git
cd mta-faq
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
